# Accessibility Pass Checklist

Manual QA pass for Dynamic Type and VoiceOver before release.

## Today

- [ ] Coach insight card: badge, title, subtitle read in order
- [ ] Quick actions: each button has a unique accessibility label
- [ ] Live activity row announces state (active / upcoming / completed)
- [ ] Date header remains legible at AX5 content size

## Coach

- [ ] Hero story title and recommendation are not duplicated by VoiceOver
- [ ] Support action chips have hint text where icons are decorative
- [ ] Health connect prompt is reachable and dismissible with VoiceOver

## Plan

- [ ] Timeline rows expose activity title, time, and status
- [ ] Drag-to-reschedule has an accessibility alternative (edit sheet)
- [ ] Delete confirmation is announced before destructive action

## Meals

- [ ] Library sections expose meal macros in summary
- [ ] Coach recommendation card links to suggested meal name
- [ ] Create-meal flow fields have labels at large content sizes

## Global

- [ ] Tab bar items have localized accessibility labels (EN + RU)
- [ ] Sheets trap focus and support rotor navigation
- [ ] No critical actions rely on color alone
